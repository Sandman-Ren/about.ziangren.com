## Code Blocks

### JavaScript Example

```javascript
// A simple React component with hooks
import { useState, useEffect } from 'react';

function Counter({ initialValue = 0 }) {
  const [count, setCount] = useState(initialValue);

  useEffect(() => {
    document.title = `Count: ${count}`;
  }, [count]);

  return (
    <div className="counter">
      <p>Current count: {count}</p>
      <button onClick={() => setCount(c => c + 1)}>
        Increment
      </button>
    </div>
  );
}

export default Counter;
```

## Lists

### Unordered Lists

Here are the key principles of good typography:

- **Readability**: Text should be easy to read with proper contrast and sizing
- **Hierarchy**: Clear visual distinction between headings and body text
- **Consistency**: Use the same styles throughout for similar elements
- **White Space**: Adequate spacing improves comprehension
  - Line height should be 1.5-1.8 for body text
  - Margins between paragraphs aid scanning
  - Indentation helps organize nested content
- **Accessibility**: Consider users with visual impairments

### Ordered Lists

Steps to implement a design system:

1. **Audit existing styles** - Document current typography and spacing
2. **Define tokens** - Create variables for colors, fonts, and spacing
3. **Build components** - Implement reusable UI components
4. **Document patterns** - Write guidelines for usage
5. **Iterate and refine** - Gather feedback and improve
   1. Conduct user testing
   2. Analyze usage patterns
   3. Update documentation
6. **Maintain consistency** - Regular reviews and updates

const typographyConfig = {

  },

Key considerations:

1. **Font loading** - Use `next/font` for optimal performance
2. **Fallbacks** - Always include system font fallbacks
3. **Variable fonts** - Reduce file size with variable font files
